# 📚 Library Management System

## 📌 Project Description

Library Management System is a DBMS mini project developed using FastAPI, MySQL, HTML, CSS, and JavaScript.
The project helps manage books, users, issuing books, returning books, and admin operations.

---

## 🚀 Features

* User Registration & Login
* Admin Login
* Add Books
* View Books
* Issue Books
* Return Books
* Leaderboard
* Role-Based Access
* MySQL Database Connectivity
* CRUD Operations

---

## 🛠️ Technologies Used

### Backend

* FastAPI (Python)

## ⚙️ How to Run

### Step 1

Install requirements

```bash
pip install -r requirements.txt
```

### Step 2

Run FastAPI Server

```bash
uvicorn main:app --reload
```

### Step 3

Open Browser

```bash
http://127.0.0.1:8000
```
